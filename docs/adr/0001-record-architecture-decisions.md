# ADR-0001: 采用架构决策记录（ADR）

- 状态：已接受
- 日期：2026-06-08

## 背景
项目有多个有长期影响的技术选择，需要可追溯地记录「为什么这么定」，
避免后续重复讨论或在不了解前因的情况下推翻。

## 决策
采用 ADR（Michael Nygard 格式）记录架构决策。
- 每个决策一个文件：`docs/adr/NNNN-标题.md`。
- 状态取值：提议 / 已接受 / 已弃用 / 被取代（注明取代者）。
- ADR 不删除；变更通过新增「取代」它的 ADR 完成。

## 结果
- 决策可追溯，新成员（含未来的 AI 协作）能快速了解上下文。
- 轻微的文档维护成本。
